# ☯️ 八字命理分析系统

专业四柱八字排盘分析工具，支持离线规则分析和在线 AI 对话。

## 🚀 在线体验

👉 **[点此打开 App](https://bazi-app.streamlit.app)**

---

## 功能特点

- **八字排盘**：四柱干支、十神、藏干、纳音、十二长生
- **命理分析**：性格、事业、财运、婚姻、健康、大运流年等 12 大方向
- **AI 对话**：接入 DeepSeek API，AI 根据经典典籍智能回答命理问题
- **历史校准**：记录人生大事，帮助提高分析准确度

## 本地运行

```bash
pip install -r requirements.txt
python run.py
```

## 独立 EXE

Windows 用户可使用打包好的 `.exe` 文件（78MB），无需安装任何环境，双击运行。

在 [Releases](https://github.com/thresh4396/bazi-app/releases) 页面下载。

## 技术栈

- 核心引擎：`lunar-python`
- 界面框架：`Streamlit`
- 打包工具：`PyInstaller`
